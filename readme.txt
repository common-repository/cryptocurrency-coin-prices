Easy to use option for setting up a bitcoin and altcoin exchange rate.

== Description ==

MyBitcoin Cryptocurrency Coin Market Prices widget is a lightweight, easy to use option for setting up a bitcoin and altcoin exchange rate option with customizable coin number amounts, prices in USD, Market Caps, 24 hour trading volumes and a weekly chart graph to show all the major stats per cryptocurrency with full shortcode support for any area or thin space within the website.

To see **live version** of cryptocurrency coin prices plugin, visit the sidebar of [bitcoinexchangeguide.com](https://bitcoinexchangeguide.com/) to see the coin market cap data display in action.

**Coin Market Cap Pricing Features**

*	Sleek, Minimal Live Coin Price Market Cap and Trading Volume Data
*	Shows US Dollar Currency Rates Table for All Top Cryptocurrencies or Specific Tokens
*	Create and Manage Beautiful Rich View Settings with Adjustable Auto-Update Intervals
*	Adaptive Font Scale Percentage for Small Sidebar and Widget Areas
*	Multi-option Coin Market Data Provider for Crypto Price Rates; CoinCap, CoinMarketCap
*	Hyperlinkable Coin Links URL Template with On/Off Option
*	Available as both Shortcode and Widget Location, No HTML Knowledge is Required

**Data providers**

The plugin retrieves a data from external cryptocurrency data sources through their API:

*	**CoinCap**

	Official site: [coincap.io](https://coincap.io). Legals: [Terms of Service](https://static.coincap.io/documents/terms_of_service.pdf), [Privacy Policy](https://static.coincap.io/documents/privacy_policy.pdf).

*	**CoinMarketCap**

	Official site: [coinmarketcap.com](https://coinmarketcap.com). Legals: [Terms](https://coinmarketcap.com/terms/), [Privacy](https://coinmarketcap.com/privacy/).

**Requirements**

*	PHP 5.4 or higher.
*	WordPress 4.0 or higher.

== Installation ==

1. Choose the plugin from the WordPress repository, or choose the plugin's archive file in 'Upload Plugin' section in WordPress 'Plugins\Add New', or upload and extract the plugin archive to the '/wp-content/plugins' directory manually.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. It will appear in the admin UI as shown in the [screenshots](http://wordpress.org/plugins/cryptocurrency-coin-prices/screenshots).
